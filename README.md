# IoUT_Topology_Control
This repository contains the source code and implementation related to the research paper:

**"Exploring QoS Metrics in SDN-based IoUT: A Comparative Study of 3D Topology Design Techniques"**

## Requirements

Before running the project, make sure the following requirements are installed on your system:

- Python version **3.0 or above**
- Operating System:
  - Linux
  - Windows

You can verify your Python installation by running:

```bash
python3 --version
```

or on Windows:

```bash
python --version
```

## Installation

Clone the repository to your local machine using:

```bash
git clone https://github.com/Rezmo2024/IoUT_Topology_Control.git
```

Navigate to the project directory:

```bash
cd IoUT_Topology_Control
```
## Create the Resutls folder
```bash
mkdir ResultsFigs
```
```bash
mkdir ResultsText
```
```bash
mkdir Throughput
```



## Running the Project

To run the code, execute:

```bash
python3 Run.py
```

For some Windows systems where `python3` is not available, use:

```bash
python Run.py
```

## Optional Dependencies

If the project requires additional Python packages and a `requirements.txt` file is available, install the dependencies using:

```bash
pip install -r requirements.txt
```
